二、命令

命令行名称： 功能：
一. whomi.exe 显示当前的用户名；

二. which Linux命令 查看Linux命令的程序文件存放在哪

三. find 起始路径 -name "被查找文件的名字" 按名字查找文件支持通配符匹配

四. find 起始路径 -size +|- n[kmg] 按文件大小查找文件 选项与参数作用：
1. +，-表示大于和小于
2. n 表示大小数据、
3. KMG表示大小单位,(kb,mb,gb)

五. grep [-n] 关键字 文件路径 通过关键字过滤文件行 选项与参数作用： 1. -n 可选 表示在结果中显示匹配的行的行数
2. 参数 关键字 必填 用” ”将关键字包围起来
3. 参数 文件路径 必填 可作为内容输入端口

六. wc [-c -m -l -w] 文件路径 统计文件的行数，单词数量 选项与参数作用： 1. -c 统计bytes数量
2. -m 统计字符数量
3. -l 统计行数
4. -w 统计单词数量
参数 文件路径 被统计的文件 可作为内容输入端口

七. | (管道符)左边命令的结果作为右边命令的输入

八. echo 参数 在命令行内输出指定内容 选项与参数作用：
1. 无需选项，一个参数用" "括起来表示要输出的内容
2. 被` `包围的内容作为命令去运行，而不是输出

九. > , >> 重定向符
1. >：将左侧命令的结果，覆盖写入到符号右侧指定的文件
2. >>：将左侧命令的结果，追加写入到符号右侧指定的文件

十. tail [-f -num] Linux路径 查看文件尾部内容，跟踪文件的最新更改 选项与参数作用：
1. 参数：Linux路径，表示被跟踪文件的路径
2. 选项，-f： 表示持续跟踪
3. 选项，-num：表示查看尾部多少行，默认10行
